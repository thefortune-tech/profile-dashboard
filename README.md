# Interactive Profile Dashboard 👤

A Flutter animated profile dashboard with gestures and smooth transitions.

## Features
- Expandable profile card with gesture tap
- Animated stats section
- Tab switcher with smooth transitions
- BLoC managing all state
- AnimationController + FadeTransition + SlideTransition

## Tech Stack
- Flutter & Dart
- BLoC (state management)
- Custom animations

## Getting Started
```bash
flutter pub get
flutter run
```
